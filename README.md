# Intern Thinkonomers

## Summer Internship Project - 2025

This website was developed during my summer internship at Thinkonomers in 2025. As part of the internship, I was tasked with creating a professional website for "The Rolling Plate," a company specializing in creative media services.

## About The Rolling Plate

The Rolling Plate offers a comprehensive range of services including:
- Advertising
- Branding
- Cinematography
- Digital Media
- Marketing
- Videography

The website serves as a portfolio and contact platform for the company, showcasing their work and services.

## Project Details

- **Duration**: Summer 2025 Internship
- **Technologies Used**:
  - HTML5
  - CSS3
  - JavaScript
- **Pages**: Home, About, Work, Services (Advertising, Branding, Cinematography, Digital Media, Marketing, Videography), Contact
